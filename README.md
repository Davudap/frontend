# -# TP_fullstack

## Pasos

1) Clonar repositorio
```bash
git clone https://github.com/Davudap/frontend
```

2) Ir al directorio de la aplicacion
```bash
cd frontend
```

3) Instalar los paquetes de la aplicacion
```bash
npm install
```

4) Dirreccion del servidor backend

```
http://localhost:8080
```
